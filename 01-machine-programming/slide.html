What is VS Code?

  • Runs multiple platforms
    • Supports running on: Windows, Linux, and macOS
  • Supports multiple languages

Or reassigning it to another value.

Variable Instantiation

                            int main() {

                            }

Type

Type

  • The int keyword stores 4 bytes interger values.
                            
                                int balance = 3000;

Type

  • To store decimals and exponentials use float and double keyword
  • The size of float is 4 bytes and the size of double is 8 bytes. Therefore, double can store more precise value
                            
                                float pi = 3.1416;
                                double c1 = 299792458;

                                double c2 = 3E8;

Type

  • The char keyword stores a single byte interger values.
  • Commonly used to store ASCII characters.
                            
                                char ch1 = 'A';
                                char ch2 = 65;

Type

  • The bool keyword stores a single byte boolean values: true or false.
                            
                                bool isRunning = true;
